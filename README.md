Este repositório reúne todos os exercícios práticos desenvolvidos durante as aulas de **Linguagem de Programação (LPR)**. Cada exercício é resolvido em **C#**, com o objetivo de fixar a lógica de programação independentemente da sintaxe da linguagem utilizada.
